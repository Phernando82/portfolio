Consulta Expediente V1.0

Consulta Expediente is a program developed in Python with the PySimpleGUI and Selenium WebDriver libraries and that can be used on computers with Windows operating system. The user fills in the fields name, NIE, number of process and year of the request and then they can save this data or consult. By clicking on the search button, the program opens automatically the page of the Ministry of Justice of Spain, fills the fields with the given data, passes the captcha and sends the query, then gets the data from process status and save to a spreadsheet named with the date of the consultation.
